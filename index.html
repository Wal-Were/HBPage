Hockey Bot for Microprocessor

Link to GitHub
ESP32-Based Bluetooth-Controlled Hockey Bot This project demonstrates the manual control of a Hockey Bot using the ESP32 microcontroller and Bluetooth. Key Features: - Remote control via smartphone app - Responsive DC motor movement for hockey gameplay - ESP32 handles Bluetooth communication and motor logic - Designed for interactive and competitive robot hockey Technologies Used: - ESP32 Microcontroller - HC-05/BT Module (or onboard ESP32 Bluetooth) - L298N Motor Driver - Arduino IDE for firmware development After testing, the Hockey Bot is capable of performing its assigned function, but its effectiveness is below competitive level and is more ideal for recreational use only.
